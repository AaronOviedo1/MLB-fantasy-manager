# MLB Fantasy Manager

Agente inteligente para optimizar lineup de MLB Fantasy Baseball.

## Features
- Análisis automático de matchups
- Verificación de probable pitchers
- Notificaciones por Telegram
- Ejecución diaria automatizada

## Configuración
Requiere variables de entorno en Railway:
- ESPN_S2
- SWID
- LEAGUE_ID
- TEAM_ID
- SEASON
- TELEGRAM_BOT_TOKEN
- TELEGRAM_CHAT_ID
